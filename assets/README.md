# Assets

This directory contains visual assets for Constraint Ranch.

## Directory Structure

```
assets/
├── screenshots/    # Game screenshots for documentation
├── sprites/        # Agent sprites and animations
├── ui/            # UI components and icons
├── backgrounds/   # Background images and textures
└── fonts/         # Custom fonts
```

## Asset Guidelines

- **Sprites**: Use SVG format for scalability
- **Screenshots**: PNG format, 1920x1080 resolution
- **Icons**: SVG format, 24x24 default size
- **Backgrounds**: WebP or PNG, optimized for web
