# Screenshots

This directory contains screenshots and visual assets for Constraint Ranch documentation.

## Directory Structure

```
screenshots/
├── gameplay/       # In-game screenshots
├── ui/            # UI component screenshots
├── agents/        # Agent species screenshots
└── puzzles/       # Puzzle example screenshots
```

## Screenshot Guidelines

- Use PNG format for screenshots
- Resolution: 1920x1080 (16:9) for gameplay
- Resolution: 1280x720 for UI components
- Include both light and dark mode variants
- Use descriptive filenames: `puzzle-spatial-001-gameplay.png`
