## Learning Objectives by Level

### Level 1-4: Ranch Hand (Tutorial & Basics)

#### Learning Objectives

| Level | Puzzle Types | Concepts Taught | Learning Outcomes |
|-------|--------------|-----------------|-------------------|
| 1 | Spatial (Tutorial) | Basic positioning, coordinate systems | Understand X/Y coordinates and basic geometric placement |
| 2 | Spatial | Coverage optimization, zone concepts | Learn about area coverage and monitoring strategies |
| 3 | Spatial | Geometric shapes (triangles, hexagons) | Understand geometric properties and exact measurements |
| 4 | Spatial | Perimeter concepts, evenly-spaced placement | Learn angular spacing and circular positioning |

#### Skills Developed

- **Computational Thinking**: Breaking down positioning problems into steps
- **Mathematical Reasoning**: Coordinate geometry, distance calculations
- **Problem Solving**: Finding optimal positions under constraints
- **Attention to Detail**: Exact positioning requirements

#### Educational Standards Alignment

- **CSTA K-12**: 1B-CS-01 (Problem decomposition)
- **Common Core Math**: 5.G.A.1 (Coordinate plane), 5.G.A.2 (Real-world problems)
- **NGSS**: 3-5-ETS1-1 (Engineering design)

### Automated Interventions

```typescript
/**
 * Automated intervention system for struggling students.
 */
const interventionRules: InterventionRule[] = [
  {
    id: 'stuck-level',
    condition: { type: 'level-stuck', threshold: 3 }, // 3 days at same level
    action: { type: 'show-hint', content: 'auto-level-hint' }
  },
  {
    id: 'failed-attempts',
    condition: { type: 'attempts-failed', threshold: 5 }, // 5 failed attempts
    action: { type: 'show-video', videoId: 'concept-review' }
  },
  {
    id: 'extended-time',
    condition: { type: 'time-spent', threshold: 30 * 60 * 1000 }, // 30 min
    action: { type: 'suggest-break' }
  },
  {
    id: 'inactivity',
    condition: { type: 'inactivity', threshold: 7 * 24 * 60 * 60 * 1000 }, // 7 days
    action: { type: 'notify-teacher' }
  }
];

function checkInterventions(student: Student): Intervention[] {
  const triggeredInterventions: Intervention[] = [];
  
  for (const rule of interventionRules) {
    const shouldTrigger = evaluateCondition(rule.condition, student);
    if (shouldTrigger) {
      triggeredInterventions.push({
        id: `intervention-${student.id}-${rule.id}`,
        studentId: student.id,
        type: rule.action.type,
        triggered: true,
        triggerCondition: rule.condition,
        action: rule.action,
        createdAt: new Date()
      });
    }
  }
  
  return triggeredInterventions;
}
```

  // Descriptions
  descriptions: {
    puzzleStart: (puzzle: Puzzle) => 
      `Starting ${puzzle.type} puzzle: ${puzzle.name}. ${puzzle.description}`;
    puzzleComplete: (score: number) => 
      `Puzzle completed! Final score: ${score} points.`;
    hintRevealed: (hint: Hint) => 
      `Hint level ${hint.level}: ${hint.text}`;
  };
}

// Screen reader announcement queue
function announce(message: string, priority: 'polite' | 'assertive' = 'polite'): void {
  const region = document.getElementById('sr-announcer');
  if (region) {
    region.setAttribute('aria-live', priority);
    region.textContent = message;
    // Clear after announcement
    setTimeout(() => { region.textContent = ''; }, 100);
  }
}
